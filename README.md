# Dhruv Bajoria - Developer Portfolio

A modern, premium, and highly interactive developer portfolio website showcasing full-stack and AI-ML development expertise.

## 🚀 Features

- **Futuristic Dark Theme** - Modern dark mode design with neon accents
- **Floating Tech Logos** - Animated technology stack icons that respond to cursor movement
- **Smooth Animations** - Micro-interactions powered by Framer Motion
- **Case Study Projects** - Detailed project showcases with problem-solution-outcome format
- **Fully Responsive** - Mobile-first design that works on all devices
- **SEO Optimized** - Built with Next.js for optimal search engine visibility
- **Accessibility Focused** - WCAG compliant with proper semantic HTML

## 🛠️ Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: React Icons

## 📦 Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd dhruv-bajoria-portfolio
```

2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## 🏗️ Build for Production

```bash
npm run build
npm start
```

## 📝 Customization

### Update Personal Information

1. **Hero Section**: Edit `components/Hero.tsx`
2. **About Section**: Edit `components/About.tsx`
3. **Projects**: Update project data in `components/Projects.tsx`
4. **Experience**: Update experience data in `components/Experience.tsx`
5. **Contact**: Update contact links in `components/Contact.tsx`
6. **Resume**: Add your resume PDF file to the `public` folder as `resume.pdf`

### Styling

- Global styles: `app/globals.css`
- Tailwind config: `tailwind.config.ts`
- Color scheme: Update neon colors in `tailwind.config.ts`

### Floating Tech Logos

Add or remove tech logos in `components/FloatingTechLogos.tsx` by modifying the `techLogos` array.
